Introduction

Base URL: http://127.0.0.1:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

Login & Get a JWT via given credentials.

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/auth/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"

    "http://127.0.0.1:8000/api/auth/login"
);

const headers = {
    "Content-Type": "application/json",

Register a new User

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/auth/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"yofszvkzbixdxtfkjfmkodabw\",
    \"email\": \"harber.jamal@example.net\",
    \"password\": \"Xp.8Ks\\\\i+#*[,n\"
}"

const url = new URL(
    "http://127.0.0.1:8000/api/auth/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "yofszvkzbixdxtfkjfmkodabw",
    "email": "harber.jamal@example.net",
    "password": "Xp.8Ks\\i+#*[,n"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example: application/json

Body Parameters

Must be at least 4 characters. Example: yofszvkzbixdxtfkjfmkodabw

password   string   

Must be at least 8 characters. Example: Xp.8Ks\i+#*[,n
